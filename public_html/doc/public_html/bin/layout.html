SYNOPSIS

Routines for the CGI programs re page layout and link formatting

layout_header(TITLE, SUBHEAD, LONGTITLE, PRETITLE)

Print to STDOUT the top part of the page HTML

TITLE

the title of the program running.

SUBHEAD

HTML for the second bar (light blue) on the page

LONGTITLE

A longer form of the title to display as a headline

PRETITLE

HTML that is put in the main content area before the first headline

layout_header(TITLE, SUBHEAD, LONGTITLE)

Print to STDOUT bottom top part of the page HTML

make_table_link(PROJECT)

Make the URL to link to the project table for PROJECT

make_list_link(ARGS)

Make the URL to link to the article list. ARGS is a hash of KEY => VALUE pairs to be sent as arguments the list program

make_log_link(ARGS)

Make the URL to link to the article list. ARGS is a hash of KEY => VALUE pairs to be sent as arguments the list program

make_article_link(NS, PAGE)

Make HTML for links to wiki pages about NS:PAGE

make_history_link(NS, PAGE, TIMESTAMP, LONG, TALKLINK)

Make the URL to link to the revision of NS:PAGE at TIMESTAMP

LONG

If this is nonzero, the timestamp will be displayed with seconds. Otherwise, only the year, month, and day are displayed in the link text.

TALKLINK

If this is nonzero, the HTML also includes a link to the revision of the corresponding talk page at TIMESTAMP

make_page_name(NS, PAGE)

Returns the formatted name of NS:PAGE, replacing the numeric NS with the text version

make_talk_name(NS, PAGE)

Returns the formatted name of the talk page corresponding to NS:PAGE. Talk pages correspond to themselves, for other pages NS is incremented by 1.

get_link_from_api(TEXT)

Parses TEXT using the API and returns the parsed HTML.

list_projects()

Returns a list reference of all projects in the database

get_cached_td_background(CLASS)

Returns the formatted HTML for a CLASS-Class table cell

get_cached_wiki_page(PAGE)

Returns the formatted HTML for PAGE on the wiki

get_cached_review_icon(REVIEWCLASS)

Gets formatted HTML for a table cell for a REVIEWCLASS article, including the icon for that REVIEWCLASS.

make_wp05_link(CAT)

Returns a link to the Wikipedia 0.5 page for review category CAT (which coult be Arts, Geography, etc.)

make_review_link(REVIEWCLASS)

Returns formatted HTML for a table cell for REVIEWCLASS
