<<

SYNOPSIS

Routines for fetching content from the wiki. This library is a wrapper that abstracts Mediawiki::API and toolserver_api.pl into a common interface and output format.

init_api()

Initialize an internal Mediawiki::API object.

pages_in_category(CATEGORY, [NS])

Returns an array reference listing pages in CATEGORY

CATEGORY _must_ be UTF-8 encoded

The NS parmater, optional, is a numeric namespace for filtering the results.

The pages in the rsulting array _do_ have the namespace prefix attached (for example Talk:Foo and Wikipedia:Bar)

The titles returned are UTF-8 encoded

pages_in_category_detailed(CATEGORY, [NS])

Returns a reference to an array of hashes, one for each page in CATEGORY.

The output format is

  {  'ns'        => NAMESPACE, 
     'title'     => TITLE,
     'pageid'    => PAGEID,
     'sortkey'   => SORTKEY,
     'timestamp' => TIMESTAMP }

CATEGORY _must_ be UTF-8 encoded.

The NS parmater, optional, is a numeric namespace for filtering the results.

The page titles in the resulting array _do_not_ have the namespace prefix attached (for example, the page Talk:Foo will show

  { 'ns' => '1',
    'title' => 'Foo',
     ...  }

The data returned is all UTF-8 encoded.

content(PAGE)

Fetch the source code of PAGE. The returned text is UTF-8 encoded.

content_section(PAGE, SECTION)

Fetch the source code of section number SECTION in PAGE. The lede is section 0.

The returned text is UTF-8 encoded.

api_namespaces()

Fetch a hash reference mapping namespace number to namespace prefix.

api_resolve_redirect(NS, TITLE)

Find the redirect target for NS:TITLE.

Return undef if NS:TITLE is not a redirect, otherwise return (DEST_NS, DEST_TITLE, REV_TIMESTAMP)

api_get_move_log(NS, TITLE)

Download a list of move log entries for NS:TITLE

Returns a reference to an array of hashes

api_edit(WIKIPAGE, CONTENT, SUMMARY)

Edit WIKIPAGE with given content and edit summary

<<
