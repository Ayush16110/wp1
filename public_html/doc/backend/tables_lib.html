SYNOPSIS

Library routines to create assessment summary tables

cached_project_table(PROJECT, [PURGE])

Get the summary table for PROJECT, using the cache and updating the cache if needed. If PURGE is nonzero then the cache will be forced to update.

Returns ($html, $wikicode).

make_project_table(PROJECT)

Create a new table for PROJECT. Does not use the cache in any way.

make_global_table()

Create a new global table. Does not use the cache in any way.

fetch_project_table_data(PROJECT)

Fetch info to create table for project. Returns a TDATA object.

make_project_table_wikicodedata(TDATA, TITLE, FORMAT)

Create wikicode for a table using data from TDATA. The overall title is TITLE. The FORMAT function is used to format the table cells. It is invoked as:

&{$FORMAT}(project, quality, importance, value)

where quality and/or importance will be undefined for the cells corresponding to the Total row and column.

get_project_categories(PROJECT)

Internal function to fetch info about assessment categories for PROJECT from the database.

get_global_table_data())

Get assessment data from the global table. Returns a TDATA object.

get_global_categories())

Internal function to Get info about global assessments from database.

cached_global_ratings_table([PURGE])

Gets the global ratings table, using the cache and updating the cache if needed. If PURGE is nonzero then the cache will be forced to update.

Returns ($html, $wikicode).

format_cell_pqi_nolink(PROJECT, QUALITY, IMPORTANCE, VALUE)

Create a formatted table cell entry. There is no link. If either QUALITY or IMPORTANCE is undefined, the cell is bold.

format_cell_pqi(PROJECT, QUALITY, IMPORTANCE, VALUE)

Create a formatted table cell entry. The entry links to the article list for PROJECT, with approprate query values preset.

get_format_cell_pqi_cat(CATEGORY, NAMESPACE)

Return a subroutine reference for tables limited to a particular category
